# Personal Data Display & Interactive Form

### What this program does
This program acts as an interactive profile display that showcases static user information and then dynamically updates and refreshes that data based on real time user inputs. It solves the problem of hardcoding temporary information by allowing the system to adapt and personalize the output for any user on the fly.

---

### How to Run It
You can run this script easily on any standard Python compiler or IDE (like PyCharm, VS Code, IDLE, or any online Python compiler). 
1. Copy the code from the `Python Exercise.py` file.
2. Paste it into your compiler.
3. Run the script and interact with the terminal prompts!
